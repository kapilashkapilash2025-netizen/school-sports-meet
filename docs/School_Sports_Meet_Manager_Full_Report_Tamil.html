1. இந்த software என்ன?

இந்த software ஒரு பள்ளி sports meet-ஐ digital-ஆ manage செய்ய உருவாக்கப்பட்ட full-stack management system. இது students register பண்ணுவது முதல் events create பண்ணுவது, participants assign பண்ணுவது, results enter பண்ணுவது, house points கணக்கிடுவது, final champion house காட்டுவது வரை முழு செயல்முறையையும் ஒரே இடத்தில் கையாள்கிறது.

இதன் முக்கிய பலம் என்னவென்றால், பல பகுதிகள் ஒரே system-ல் இணைக்கப்பட்டுள்ளன: மாணவர் தகவல், event structure, scoring logic, result history, projector mode, leaderboard, house overview, admin access control, மற்றும் theme-based presentation.

முக்கிய நோக்கம்

  • Sports meet data-ஐ paper-free digital முறையில் manage செய்வது
  • House points கணக்கீட்டை தானியங்கியாக்குவது
  • Admin வேலைகளை வேகமாகவும் குறைந்த பிழையுடனும் நடத்துவது
  • Final results மற்றும் projector presentation-ஐ அழகாக காட்டுவது
  • School identity-ஐ branding உடன் software-ல் பிரதிபலிப்பது

  • athletics_results: athletics-based house points support

12. Main Program Logic

Student Logic

  • Student ID user manually set செய்ய முடியாது
  • Gender அடிப்படையில் suffix generate ஆகும்: B அல்லது G
  • Format: S0001B / S0001G
  • Duplicate check: பெயர் + பிறந்த தேதி
  • Grade missing இருந்தாலும் birth year மூலம் resolve செய்யலாம்

Event Logic

  • Official events load செய்யலாம்
  • Event creationக்கு age, gender, scoring mode validation
  • Participant duplicate protection
  • Event list display order pre-defined sports order அடிப்படையில் வருகிறது

Result Logic

  • Selected event scoring mode backend-ல் verify செய்யப்படும்
  • All participants result set complete ஆக வேண்டும்
  • Incomplete result save செய்ய முடியாது
  • Event completed status update செய்யப்படும்

Leaderboard Logic

  • Official results points + manual event points + athletics points + extra manual points அனைத்தும் total ஆகும்
  • Race-type manual event chart totals leaderboard chart breakdown-ல் exclude செய்யப்பட்டுள்ளது
  • Dense rank மூலம் champion order தீர்மானிக்கப்படுகிறது
